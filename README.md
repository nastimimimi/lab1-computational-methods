Git та Linux CLI

## Модель
Автоматизоване складання розкладу занять кафедри — Генетичний алгоритм (5 семестр)

## Автор
Баранова Анастасія, група АІ-231

## Опис
У роботі розроблено програмну систему для автоматичного формування розкладу занять кафедри. В основі рішення лежить **генетичний алгоритм**, доповнений **локальним пошуком (2-opt)**.
Цільова функція спрямована на мінімізацію:
- Накладань у використанні ресурсів (викладачі, аудиторії, групи)
- Порожніх вікон у розкладі викладачів
- Розходжень між реальним розкладом і побажаннями викладачів щодо зручного часу
  
### Параметри системи
| Параметр | Значення |
|---|---|
| Викладачів | 15 |
| Дисциплін | 30 |
| Аудиторій | 12 |
| Груп | 10 |
| Часових слотів | 30 (5 днів × 6 пар) |

### Алгоритм
- **Метод:** Генетичний алгоритм + Локальний пошук (2-opt)
- **Розмір популяції:** 100
- **Максимум ітерацій:** 1000
- **Селекція:** Турнірна
